Ethereum-powered incentivized deposit-refund system for bottled beverages in Germany.

### Setup 
1. `git clone <url>`
2. `npm install`

### Dependencies
- [Truffle](https://truffleframework.com/)
- [OpenZeppelin Solidity](https://github.com/OpenZeppelin/openzeppelin-solidity)
